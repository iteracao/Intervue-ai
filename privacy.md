# Privacy Policy - Intervue AI

Last updated: 2026-03-31

## Overview
Intervue AI is a desktop application that provides real-time transcription, question detection, and AI-powered answer suggestions during live conversations.

## What the app processes
- System audio output
- Speech transcriptions
- User configuration settings
- User-provided API credentials
- Generated AI suggestions

## What the app does not process
- Microphone input
- Hidden background capture after detection is stopped

## How data is processed
Intervue AI primarily operates locally on the user's device.

When the app is actively used, audio or transcription content may be transmitted to OpenAI for transcription or answer generation.

## What is stored locally
- Application settings
- Session logs
- Optional debug artifacts if explicitly enabled by the user

API keys are stored securely using Windows Credential Manager.

## User control
- Detection runs only when explicitly started by the user
- Detection can be stopped at any time
- Optional debug artifacts can be enabled or disabled by the user

## Third-party services
OpenAI may process submitted audio or text according to its own terms and privacy policies when the app sends requests for transcription or answer generation.
