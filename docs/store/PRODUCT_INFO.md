# Intervue AI - Product Information

## Product Name
Intervue AI

## One-line Summary
Real-time desktop interview assistant that processes live conversations, detects questions, and suggests concise AI-powered answers.

## Category
Primary: Productivity
Secondary: Developer Tools

## Target Audience
- Software engineers
- Technical professionals
- Job candidates preparing for interviews
- Users who want real-time assistance during live conversations

## Core Value Proposition
Intervue AI helps users during live technical conversations by converting speech into text, detecting likely questions, and generating concise suggested answers in real time.

## Core Capabilities
- Real-time system audio processing
- Live transcription
- Automatic question detection
- Context-aware AI answer suggestions
- Multiple transcription providers
- Secure local credential storage
- Desktop-first workflow

## Supported Providers
- OpenAI
- Vosk
- Faster-Whisper

## Supported Languages
- English
- Portuguese

## Privacy Positioning
- Local-first desktop app
- Detection runs only when explicitly started by the user
- No hidden background recording or background data collection
- API keys stored securely in Windows Credential Manager
- Audio and text may be sent to third-party AI services only when user-configured providers require it

## Important Disclosure
Intervue AI processes system audio output to provide transcription and AI assistance during active user sessions.
