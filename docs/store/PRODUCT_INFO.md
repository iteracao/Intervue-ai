# Intervue AI - Product Information

## Product Name
Intervue AI

## One-line Summary
Real-time desktop interview assistant that processes live conversations, detects questions, and suggests concise AI-powered answers.

## Category
Primary: Productivity
Secondary: Developer Tools

## Target Audience
- Software engineers
- Technical professionals
- Job candidates preparing for interviews
- Users who want real-time assistance during live conversations

## Core Value Proposition
Intervue AI helps users during live technical conversations by converting speech into text, detecting likely questions, and generating concise suggested answers in real time.

## Core Capabilities
- Real-time system audio processing
- Live transcription
- Automatic question detection
- Context-aware AI answer suggestions
- OpenAI transcription
- Secure local credential storage
- Desktop-first workflow

## AI Services
- OpenAI

## Current Language Scope
- Best current results are in English and Portuguese
- Transcription is automatic per spoken chunk
- Question detection quality may vary outside the app's currently tuned interview patterns

## Privacy Positioning
- Local-first desktop app
- Detection runs only when explicitly started by the user
- Captures system audio output, not microphone input
- No hidden background recording or background data collection
- API keys stored securely in Windows Credential Manager
- Audio and text may be sent to OpenAI during active configured sessions

## Important Disclosure
Intervue AI processes system audio output during active user-started sessions to provide transcription, question detection, and AI answer suggestions.

## What The App Does Not Do
- It does not capture microphone input
- It does not run hidden background recording when detection is stopped
- It does not include offline/local transcription providers in the current packaged app
- It does not send audio or transcript data anywhere except OpenAI requests made during active use
