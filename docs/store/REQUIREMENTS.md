# Requirements - Intervue AI

## Operating System
- Windows 10
- Windows 11

## Hardware
- Audio output device
- Moderate CPU recommended for real-time transcription workloads
- Higher CPU/RAM recommended for local transcription models such as Faster-Whisper

## Network
- Internet connection required for OpenAI-based features
- Not required for fully local transcription configurations

## Optional External Dependencies

### OpenAI mode
- Valid OpenAI API key

### Vosk mode
- Downloaded Vosk model available locally

### Faster-Whisper mode
- Faster-Whisper executable available locally
- Corresponding model available locally
