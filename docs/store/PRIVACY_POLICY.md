# Privacy Policy - Intervue AI

Last updated: 2026-03-30

## Overview
Intervue AI is a desktop application that provides real-time transcription, question detection, and AI-powered answer suggestions during live conversations.

## What the app processes
The app may process:
- System audio output
- Speech transcriptions
- User configuration settings
- User-provided API credentials
- Generated AI suggestions

## How data is processed
Intervue AI primarily operates locally on the user's device.

Depending on the configured provider, audio or transcription content may be transmitted to third-party AI services such as OpenAI for transcription or answer generation.

## What is stored locally
The app may store locally:
- Application settings
- Session logs
- Optional debug artifacts if explicitly enabled by the user

API keys are stored securely using Windows Credential Manager.

## What is not collected
Intervue AI does not perform hidden telemetry or undisclosed background data collection.

## User control
Users control:
- Whether to start or stop live detection
- Which providers are configured and used
- Whether optional debug artifacts are saved

## Third-party services
If the user enables cloud-based AI providers, those services may process submitted audio or text according to their own terms and privacy policies.
