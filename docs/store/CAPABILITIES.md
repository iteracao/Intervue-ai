# Capabilities - Intervue AI

## Package capabilities
- internetClient
- privateNetworkClientServer
- runFullTrust

## Why they are needed

### internetClient
Required for cloud-based AI features such as OpenAI transcription and answer generation.

### privateNetworkClientServer
Useful for standard desktop networking scenarios.

### runFullTrust
Required because Intervue AI is a packaged desktop application that performs full desktop operations such as system audio processing and optional external tool integration.

## Reviewer note
Intervue AI is a desktop productivity application. It processes system audio output locally to provide transcription and AI assistance during active user sessions. It does not perform hidden recording or background capture.
