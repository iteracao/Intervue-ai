Intervue AI is a desktop assistant for live technical conversations and interviews.

It processes system audio output locally, transcribes speech in real time, detects likely questions, and suggests concise AI-powered answers.

Key features:
- Real-time system audio processing
- Live transcription
- Automatic question detection
- Context-aware answers
- Multiple transcription providers
- Secure API key storage

User control:
- Runs only when explicitly started
- Can be stopped anytime
- No background recording

Privacy:
- Local-first
- No hidden data collection
- External AI only when configured
